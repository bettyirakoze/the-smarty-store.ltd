# 🛒 SmartStore Inventory Management System

## 📌 Project Description
SmartStore is a simple browser-based inventory management system built using **HTML, CSS, and JavaScript**.  
It allows users to manage product data without a backend or database. All data is stored in a JavaScript array.

This project was developed as a junior web developer assignment for SmartStore Ltd.

---

## 🚀 Features

- 📊 Dashboard with product statistics
- ➕ Add new products
- ✏️ Update product quantity and price
- ❌ Delete products with confirmation
- 🔍 Search products by ID, name, or category
- 📋 Dynamic inventory table (auto-updates)
- 🧾 Input validation (no empty or invalid values)
- 🧠 In-memory data storage (JavaScript array only)
- 📱 Responsive design for mobile and desktop
- 🎉 Welcome screen before entering dashboard

---

## 🧑‍💻 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
